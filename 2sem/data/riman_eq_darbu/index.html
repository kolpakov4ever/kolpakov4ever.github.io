Замечания о взаимном расположении интегральных сумм

  1. Существует лишь два варианта взаимного расположения $\sup s_* $ и $\inf S^*$:
    • $\sup s_* < \inf S^*$ – критерий Дарбу не выполнен
    • $\sup s_* = \inf S^*$ – критерий Дарбу выполнен
    • Вариант $\sup s_* > \inf S^* $ невозможен, т.к. иначе найдётся такая пара сумм, что $s_{* (i)} = S_{(j)}^* $, а это невозможно
  2. Для одного и того же разбиения: $\sup s_* \leqslant \Sigma_R \leqslant \inf S^* $, где $\Sigma_R$ – риманова сумма

Лемма об эпсилон-переходе к критерию Дарбу

Хотим доказать:

Если для $\forall \varepsilon>0$ найдётся такое разбиение $\{x_i\}$ отрезка $[a ; b]$, что $$S^*\{x_i\} - s_*\{x_i\}| < \varepsilon$$ где $s_*$ и $S^*$ – нижняя и верхняя сумма Дарбу по заданной функции, то критерий Дарбу выполнен.

Риман -> Дарбу

Имеем:

По определению Римана $ \forall \varepsilon_0>0\text{ } \exists$ такое разбиение $\{x_i\}_{k=1}^n$, что

$\text{Для } \forall \{\xi_k\}_{k=1}^n \text{, где }$

$x_{k-1} \leqslant \xi_k \leqslant x_k$

$\text{Верно неравенство:}$

$|A - (\sum_{i=1}^{n} f(\xi_i) (x_i - x_{i-1}))| < \varepsilon_0$

Хотим доказать:

Для любого $\varepsilon > 0$ найдётся такое разбиение $\{\eta_i\}$, что $S^* \{\eta_i\} - s_* \{\eta_i\} < \varepsilon$.

Более того, мы утверждаем, что это и будет разбиение $\{x_i\}_{k=1}^n$.

Итак,

  • Фиксируем $\varepsilon$
  • Выбираем $\varepsilon_0 := \varepsilon/4$
  • Условия интегрируемости Римана выполняются для любого оснащения. Выберем на каждом промежутке $[x_{i-1}; x_i]$
    • Такое $\xi_i$, что $\xi_i < m_i + \frac{\varepsilon_0 }{b-a}$, где $m_i = \inf f$ на данном промежутке
    • Такое $\zeta_i$, что $\zeta_i > M_i - \frac{\varepsilon_0 }{b-a}$б где $M_i = \sup f$ на данном промежутке

Такие точки существуют по свойству точных граней (которое легко доказывается от противного).

//TODO В этом месте должна быть ссылка на конспект по точным граням, который ещё не оформлен.

Тогда

$\sum_{i=1}^{n} f(\xi_i) (x_i - x_{i-1}) < $

$\sum_{i=1}^{n} m_i (x_i - x_{i-1}) + $

$\frac{\varepsilon_0 }{b-a} \sum_{i=1}^{n}(x_i - x_{i-1})$

$\sum_{i=1}^{n} f(\xi_i) (x_i - x_{i-1}) < $

$s_* \{x_i\} + \varepsilon_0 $

С другой стороны

$\sum_{i=1}^{n} f(\zeta_i) (x_i - x_{i-1}) > $

$\sum_{i=1}^{n} M_i (x_i - x_{i-1}) - $

$\frac{\varepsilon_0 }{b-a} \sum_{i=1}^{n}(x_i - x_{i-1})$

$\sum_{i=1}^{n} f(\xi_i) (x_i - x_{i-1}) > $

$S^* \{x_i\} - \varepsilon_0 $

Обозначим римановую сумму по $\xi$ за $R_\xi$, а по $\zeta$ за $R_\zeta$, и сложим два неравенства: $$R_\xi + S^* \{x_i\} - \varepsilon_0 < R_\zeta + s_* \{x_i\} + \varepsilon_0 $$ $$ S^* \{x_i\} - s_* \{x_i\} < R_\zeta - R_\xi + 2\varepsilon_0 $$

Т.к. $$|A - R_\xi| < \epsilon_0$$ $$|A - R_\zeta| < \epsilon_0$$ Очевидно $$|R_\xi - R_\zeta| < 2\epsilon_0$$ $$ -2 \epsilon_0 < R_\xi - R_\zeta < 2 \epsilon_0$$

Отсюда $$S^* \{x_i\} - s_* \{x_i\} < 4\varepsilon_0$$ $$S^* \{x_i\} - s_* \{x_i\} < \varepsilon$$

Ч.Т.Д. █

Лемма о достаточной мелкости

//TODO Данный раздел очень сложен для понимания без рисунков. Мы недеемся, что скоро наш художник вернётся к работе и нарисует всё, что только можно. Пока же приходится довольствоваться тем, что есть ;c

Имеем:

$$\sup s_* = \inf S^* = A$$

Хотим доказать:

$ \forall \varepsilon > 0 \text{ } \exists \delta > 0$, что лишь только для разбиения $\{\psi_i\}$ выполнено $\lambda \{\psi_i\} < \delta $, так сразу $A - s_* \{\psi_i\} < \varepsilon $ и $S^* \{\psi_i\} - A < \varepsilon$. Для этого необходимо найти зависимость $\delta = \delta(\varepsilon)$

Итак,

  • Заметим – существует хотябы одно разбиение $\{x_i\}$, для которого верхняя и нижняя суммы попадают в эпсилон-окрестность $A$. Это следует из свойств точных граней.
  • Обозначим $p_0 := s_* \{x_i\} - (A-\varepsilon)$ и $P_0 := (A+\varepsilon) - S^* \{x_i\} $ – расстояния от сумм этого разбиения до границ эпсилон-окрестности
  • Для инфинумов и супремумов введём классические обозначения – $m_i^{(x)}$ и $M_i^{(x)}$
  • Пусть нам теперь дано разбиение $\{y_i\}_{i=0}^N$. Для его инфинумов и супремумов введём обозначения $m_i^{(y)}$ и $M_i^{(y)}$. Рассмотрим k-й его промежуток этого разбиения:
    • Если $[y_{k-1}; y_k]$ целиком $\in [x_{i-1}; x_i]$, то $m_k^{(y)} \geqslant m_i^{(x)}$. В самом деле, если предположить обратное, то $m_i^{(x)}$ перестанет быть инфинумом всего большего промежутка. В свою очередь, $M_k^{(y)} \leqslant M_i^{(x)}$. Наличие таких промежутков не может уменьшить нижнюю сумму $\{y_i\}$ или увеличить верхнюю относительно сумм разбиения $\{x_i\}$.
    • В другом же случае отрезок $[y_{k-1}; y_k]$ пересекается с несколькими промежутками разбиения по $x$. Для простоты озаботимся тем, чтобы $[y_{k-1}; y_k]$ не мог содержать в себе ни один промежуток разбиения по $x$ целиком – пусть $\lambda \{y_i\} < \delta < \max_{i=1}^n (x_i - x_{i-1})$. Теперь мы знаем, что $[y_{k-1}; y_k]$ включает в себя лишь одну точку начального разбиения – $x_j$. Тогда $$m_k^{(y)} \geqslant \min (m_j^{(x)}, m_{j+1}^{(x)})$$ $$M_k^{(y)} \leqslant \min (M_j^{(x)}, M_{j+1}^{(x)})$$

Заметим, что это значит, что на одном из промежутков $[y_{k-1}; x_j]$ или $[x_j; y_k]$ значение инфинума понизилось, и на каком-то значение супремума возросло. Это нежелательное для нас поведение – мы не хотим, чтобы было $p:= s_* \{x_i\} - s_* \{y_i\} \geqslant p_0$ или $P: = S^* \{y_i\} - S^* \{x_i\} \geqslant P_0$, т.к. тогда суммы разбиения $\{y_i\}$ перестанут принадлежать нужной эпсилон-окрестности. К счастью, «потери» для на k-м промежутке

  • Для нижних сумм: $p_k \leqslant (y_k - y_{k-1}) * \Delta_k^{inf}$, где $\Delta_k^{inf} := |m_j^{(x)} - m_{j+1}^{(x)}|$
    • Т.к. $ (y_k - y_{k-1}) \leqslant \delta$, общие потери $p \leqslant \delta \sum_{i=1}^{n-1} \Delta_k^{inf}$. Мы хотим $p < p_0$, тогда $\delta < \delta_1 < \frac{p_0}{\sum_{i=1}^{n-1} \Delta_k^{inf}}$
  • Для верхних сумм: $P_k \leqslant (y_k - y_{k-1}) * \Delta_k^{sup}$, где $\Delta_k^{sup} := |M_j^{(x)} - M_{j+1}^{(x)}|$
    • Т.к. $ (y_k - y_{k-1}) \leqslant \delta$, общие потери $P \leqslant \delta \sum_{i=1}^{n-1} \Delta_k^{sup}$. Мы хотим $P < P_0$, тогда $\delta < \delta_2 < \frac{P_0}{\sum_{i=1}^{n-1} \Delta_k^{sup}}$

Таким образом, искомая дельта:

$$ \delta := \frac{1}{\pi} \min \{$$

$$\frac{s_* \{x_i\} - (A-\varepsilon)}{\sum_{i=1}^{n-1} |m_i^{(x)} - m_{i+1}^{(x)}|}$$

$$\text{ ;; }$$

$$\frac{(A+\varepsilon) - S^* \{x_i\}}{\sum_{i=1}^{n-1} |M_i^{(x)} - M_{i+1}^{(x)}|}$$

$$\text{ ;; }$$

$$\max_{i=1}^n (x_i - x_{i-1})$$

$$\}$$

Где разбиение $\{x_i\}$ также зависит от $\varepsilon$. Деление на $\pi$ – т.к. нам необходимо число меньшее, чем значение минимума.

Дарбу -> Риман

Имеем:

$$\sup s_* = \inf S^* = A$$

Хотим доказать:

$$ \forall \varepsilon>0\text{ } \exists \text{ } \exists \delta=\delta(\varepsilon)>0 :$$

$\text{Для } \forall \{\xi_k\}_{k=1}^n \text{, где }$

$x_{k-1} \leqslant \xi_k \leqslant x_k$

$\text{Верно неравенство:}$

$|A - (\sum_{i=1}^{n} f(\xi_i) (x_i - x_{i-1}))| < \varepsilon_0$
