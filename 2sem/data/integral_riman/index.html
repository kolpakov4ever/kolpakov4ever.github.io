Интеграл по Риману

Введение

Несмотря на то, что весь курс школьной геометрии посвящён работе с площадями, вероятно, каждый из слушателей затруднится дать строгий ответ на вопрос – что же такое площадь? Вопрос это очень сложный, и окончательный ответ на него дал только Лебег, обобщив интеграл Римана. Интересный факт заключается в том, что, как оказалось, родственное понятие для площади – вероятность.

Чтобы хоть на что-то опереться, мы можем сходу дать своё определение для площади прямоугольника – это есть произведение его длины и ширины. Теперь мы можем попытаться свести площади сложных фигур к площадям прямоугольника. В этом нам и поможет интеграл Римана.

Риманова сумма

Мы хотим найти площадь под графиком некоторой функции $f$ на промежутке $[a;b]$. Разделим этот промежуток на произвольные части с помощью $n$ границ вида $$(x_0 = a) < x_1 < … < x_{n - 1} < (x_n = b)$$ Такое множество $\{x_k\}$ – разбиение.

Введём оснащение этого разбиения. Это множество $\{ \xi_k \}_{k=1}^n$, таких, что $x_{k-1} \leqslant \xi_k \leqslant x_k$, иными словами, каждая точка $\xi$ взята произвольно на одном из промежутков.

Тогда Риманова (интегральная) сумма $\sigma_R := \sum_{i=1}^{n} f(\xi_i) (x_i - x_{i-1})$.

Визуально это сумма площадей прямоугольников, где $f(\xi_i)$ – высота прямоугольника, а $(x_i - x_{i-1})$ – ширина. Несложно заметитить, что такая совокупность прямоугольников будет заметно отличаться от фигуры под графиком. Как же приблизить Риманову сумму к желаемому результату?

Очевидное предположение, что нужно устремить $n \to \infty$. Оно не верно. Так, если $\forall i \in [0;n-1] x_i \in [a; \frac{a+b}{2}]$, очевидно, что даже при $n \to \infty$ мы нисколько не приблизимся к подобию фигуры под графиком.

Значит, нам нужно потребовать, чтобы точки распределялись по промежутку более равномерно. Введём для этого специальную характеристику – мелкость разбиения: $$\lambda \left( \{ x_k \}_{k=0}^n \right) := max_{i=1}^n (x_i - x_{i-1})$$ Если мелкость разбиения будет стремиться к нулю, то

  • Количество точек будет необходимо стремиться к бесконечности
  • Точки будут распределяться по промежутку достаточно равномерно

Определение интеграла

В упрощённом виде определение может быть записано как $$ \int_{a}^{b} f(x)dx := \lim_{\lambda \left( \{ x_k \}_{k=0}^n \right) \to 0} (\sum_{i=1}^{n} f(\xi_i) (x_i - x_{i-1}))$$ Эта запись не является достаточно строгой по двум прияинам:

  • Она умалчивает о происхождении наборов $\{x_k\}$ и $\{\xi_k\}$
  • Значение суммы не зависит напрямую от мелкости разбиения. Несложно привести пример двух сумм Римана, которые будут иметь разные значения при одинаковой мелкости.

Более точное определение:

$$ \int_{a}^{b} f(x)dx = A :\Leftrightarrow$$ $$ \forall \epsilon>0 \text{ } \exists \delta=\delta(\epsilon)>0 :$$ $$\text{Для } \forall \{x_k\}_{k=1}^n \text{ такого, что } \forall k\text{ }x_k \in [a;b] \text{ и } \lambda\{x_k\} < \delta$$ $$\text{И для } \forall \{\xi_k\}_{k=1}^n \text{, где } x_{k-1} \leqslant \xi_k \leqslant x_k$$ $$\text{Верно неравенство:}$$ $$|A - (\sum_{i=1}^{n} f(\xi_i) (x_i - x_{i-1}))| < \epsilon$$

Интеграл и есть площадь фигуры под графиком на промежутке $[a;b]$.

Зачем нужно оснащение

Вероятно, этот вопрос мог заинтересовать слушателя, проходившего интегралы в школе, где в качестве оснащения выбираются для определённости всегда правые границы. Требование «условие должно выполняться для любого выбранного оснащения» может серьёзно усложнять работу с определением. Тем не менее, это требование необходимо, чтобы некоторые «плохие» функции, у которых значение суммы Римана от выбора оснащения меняется просто не имели суммы по определению. Допущение существования у них суммы может привести к парадоксальным (и неверным) выводам.

Непротиворечивость

Для полной уверенности в том, что мы всё делаем верно, нужно доказать, что новое определение площади не противоречит старому, а именно – с помощью определения Римана можно посчитать площадь прямоугольника, и она будет совпадать со значением, полученным по формуле «длина на ширину». Данное задание предлагалось для самостоятельного решения на тесте в виде интегрирования константной функции. Ниже предлагается разбор этого задания в общем виде.

$$ \int_a^b C * dx = \sum_{i=1}^{n} f(\xi_i) (x_i - x_{i-1}) = (1)$$ $$ \text{Так как } \forall x \text{ } f(x)=C $$ $$ (1) = \sum_{i=1}^{n} C (x_i - x_{i-1}) = C * \sum_{i=1}^{n} (x_i - x_{i-1}) = (2)$$ $$ \text{Рассмотрим полученную сумму: } \sum_{i=1}^{n} (x_i - x_{i-1}) = $$ $$ (x_0 - x_1) + (x_1 - x_2) + … + (x_{n-2} - x_{n-1}) + (x_{n-1} - x_n) = $$ $$ (x_0 + x_1 + … + x_{n-2} + x_{n-1}) - (x_1 + … + x_{n-2} + x_{n-1} + x_n) = $$ $$ x_0 + (x_1 - x_1) + … + (x_{n-1} - x_{n-1}) - x_n = a - b$$ $$ \text{ Т.е. } (2) = C * (a-b)$$

Здесь $(a-b)$ – ширина прямоугольника, а $C$ – его высота. Ч.Т.Д.
