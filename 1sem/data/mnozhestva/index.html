Множества и операции над ними. Формулы де Моргана.

Базовые понятия

  • Множеством называется совокупность некоторых элементов; множество может быть задано с помощью перечисления или с помощью определяющего свойства: $A = \{1, 2, 7, 4 \}$ или $Y = \{y~|~y \in \mathbb N ~ \text{и} ~ y < 7\}$
    Последнее выражение обозначает, что $y \in Y \Leftrightarrow (y \in \mathbb N ~ \text{и} ~ y < 7)$, т.е. «если $y$ принаддежит натуральным числам, и $y$ меньше семи, это равносильно тому, что $y$ принадлежит $Y$».
  • Как уже можно было догадаться, если элемент $x$ входит в множество $X$, то записывают $x \in Х$ (принадлежит).
  • Если множество $X$ является частью множества $Y$, то записывают $X \subset Y$ (содержится).

Нельзя писать $X \in Y$ для множеств одного порядка и не любое свойство может задавать множество – иллюстрацией этого является парадокc Брадобрея.

Со строгостью математических обозначений для знаков $ \subset, \subseteq, \subsetneq $ существует некоторая неопределённость. Смотрите статью «Неопределённость математических терминов и обозначений».

Операции над множествами

  • Равенство. $A = B$ если они состоят из одних и тех же элементов.
  • Объединение. Если $A = \{1, 2\}, B = \{10, 20\}, C = A \cup B$, то $C \{c ~| c \in A\text{ или }c \in B\}$, т.е. $C = \{1, 2, 10, 20\}$. Объединение коммутативно и ассоциативно, т.к. определяется через коммутативный и ассоциативный логический оператор «или».
  • Пересечение. Если $A = \{3, 6, 9 ,12\}, B = \{2, 4, 6, 8, 10, 12\}, C = A \cap B$, то $C \{c ~| c \in A\text{ и }c \in B\}$, т.е. $C = \{6, 12\}$. Пересечение коммутативно и ассоциативно, т.к. определяется через коммутативный и ассоциативный логический оператор «и».
  • Отрицание (дополнение). Если $A = \{a ~| a \in N \text{ и } a > 3\}, C = (\neg A) \cup N$, то $C \{c ~| c \notin A\text{ и }c \in N\}$, т.е. $C = \{1, 2, 3\}$
  • Разность. Если $A = \{ 1, 2, 3, 4, 5, 6 \}, B = \{2, 4, 6\}, C = A \setminus B$, то $C \{c ~| c \in A\text{ и }c \notin B\}$, т.е. $C = A \cap (\neg B)$, т.е. $C = \{1, 3, 5\}$. Дополнение не коммутативно и не ассоциативно. Примеры: $$A = \{1, 2, 3\}, B = \{2, 3, 4\} \text{, тогда } A \setminus B = \{1\} \text{, но } B \setminus A = \{4\}.$$ $$C = \{3, 4, 5\} \text{, тогда } A \setminus (B \setminus C) = \{1, 2, 3\} \setminus \{2\} = \{1, 3\} \text{, но } (A \setminus B) \setminus C = \{1\} \setminus \{2, 3, 4\} = \{1\}$$

Законы де Моргана

Законы де Моргана – два правила, которые связывают операции объединения и пересечения при помощи логического отрицания.

  1. Отрицание объединения (конъюнкции) есть не что иное, как пересечение (дизъюнкция) отрицаний: $ \neg ( \cup_{i=0}^n A_i) = \cap_{i=0}^n (\neg A_i) $
    Докажем это. Все преобразования по определению:
    $ x \in \neg (\cup_{i=0}^n A_i) \Rightarrow x \notin \cup_{i=0}^n A_i \Rightarrow \forall i \text{ } x \notin A_i \Rightarrow \forall i \text{ }x \in \neg A_i \Rightarrow x \in \cap_{i=0}^n (\neg A_i) $
